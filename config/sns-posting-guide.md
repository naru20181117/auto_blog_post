## 共通原則：全プラットフォームに適用

### 最重要ポイント

| 要素 | 詳細 |
|------|------|
| **最初の3秒** | 視聴者の60%以上が3秒以内に離脱。強烈なフック必須 |

| **字幕必須** | 音声オフ視聴が多く、字幕で視聴完了率向上 |

## X (Twitter)

### アルゴリズムのポイント制

| アクション | スコア倍率 |
|-----------|-----------|
| リプライへの返信（会話成立） | **75x** |
| リツイート | 20x |
| 引用ポスト | 15x |
| リプライ | 13.5x |
| ブックマーク | 10x |
| いいね | 1x（基準値） |

### 投稿ルール

#### 推奨

- 画像・動画をネイティブ投稿（40%エンゲージメント向上）
- 動画は**30〜90秒**、縦型推奨
- ハッシュタグは**1〜2個**
- スレッド形式（3倍のエンゲージメント）
- 投稿後15分以内にリプライに返信

#### 禁止・注意

- **外部リンク**: 50-90%リーチ減（特に非Premium）
- リンクを貼る場合はリプライに配置
- ハッシュタグ3個以上で減点リスク

### 投稿時間

- 平日: 8-10時、19-21時
- 週末: 9-11時
- 避ける: 15-17時

### X Premium

- **2〜4倍**のリーチブースト
- 真剣に運用するなら加入推奨

## Threads

### アルゴリズムの特徴

- **会話重視**: いいねより「返信」が圧倒的に重要
- フォロワー数よりも「会話を生むコンテンツ」が優遇
- 投稿後30〜90分が勝負

### 投稿ルール

#### 推奨

- 意見を求める・質問する投稿
- 自分のコメントに即レス
- トピックタグを活用（SEO的効果）
- 本音・オーセンティックなトーン
- 毎日投稿で継続的な露出

#### 禁止・注意

- 投稿して放置（コメントに返さない）
- Instagramと同じ写真メインの投稿
- 堅すぎるビジネストーン
- フック（導入）のない投稿

### プラットフォーム特性

- **4億人以上**の月間ユーザー（2025年中頃）
- テキスト・会話主体のプラットフォーム
- 写真よりも「話題になるネタ」が伸びる

## Instagram

### 2025-2026 アルゴリズムの3大シグナル

| ランク | シグナル | 説明 |
|--------|----------|------|
| 1 | **視聴時間** | 最初の3秒を超えて視聴されるか |
| 2 | **いいね率** | リーチに対するいいねの割合 |
| 3 | **送信（シェア）** | DMでのシェアが**3-5倍**重要 |

### Reels の最適化

| 項目 | 推奨値 |
|------|--------|
| 長さ | **15〜45秒**（30秒前後が理想） |
| 3秒維持率 | 60%以上を目指す |
| ハッシュタグ | **5個以内**、関連性重視 |
| キャプション | キーワードを含む（SEO効果） |

### 投稿ルール

#### 推奨

- カルーセル投稿（滞在時間UP）
- Reels で新規リーチ獲得
- ストーリーズで日常的な接点維持
- DMシェアされやすい「保存したくなる」コンテンツ
- オリジナルコンテンツ（転載は減点）

#### 禁止・注意

- エンゲージメントベイト（「いいねで〇〇」等）
- 90秒以上のReels（視聴完了率低下）
- 過度なハッシュタグ（スパム判定）
- 他人のコンテンツの無断転載

### 投稿後24〜48時間

- この期間のエンゲージメントが最も重要
- 投稿直後にストーリーズで告知すると効果的

## ショート動画 共通ベストプラクティス

### 最初の3秒（フック）の作り方

| テクニック | 例 |
|-----------|-----|
| 衝撃的な映像 | 予想外のシーン、ビフォーアフター |
| 質問で始める | 「〇〇って知ってた？」 |
| 数字を出す | 「3つの方法」「90%の人が知らない」 |
| 禁止・否定 | 「絶対やるな」「〇〇は間違い」 |
| 好奇心ギャップ | 「最後まで見て」「結末が衝撃」 |

### プラットフォーム別 最適動画長

| プラットフォーム | 推奨長さ |
|-----------------|---------|
| TikTok | 15〜30秒 |
| Instagram Reels | 15〜45秒 |
| YouTube Shorts | 15〜35秒 |
| X (Twitter) | 30〜90秒 |

### 視聴完了率を上げるコツ

1. **ループ構造**: 終わりが始まりにつながる
2. **テンポ**: カット数を増やす（2〜3秒ごと）
3. **字幕**: 音声と同期したテキスト
4. **音楽**: トレンドの音源を使用
